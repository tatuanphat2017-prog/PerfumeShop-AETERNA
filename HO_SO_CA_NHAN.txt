TECH STACK QUEN DÙNG
- Backend:   ASP.NET Core (C#) — Web API, phân lớp Controller/
             Service/Data/Model, dùng DTO.
- Database:  SQL Server + Entity Framework Core (Code First,
             Migrations).
- Auth:      JWT + BCrypt, phân quyền Admin/User.
- Frontend:  HTML/CSS/JS thuần (gọi API bằng fetch), responsive.
- IDE:       Visual Studio 2022.
- Deploy:    Hosting free (đã dùng MonsterASP.NET) qua Web Deploy.
